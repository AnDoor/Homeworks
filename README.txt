El siguiente programa ayuda a ordenar a base de una pila los datos basicos de un trabajador dentro de una empresa.
Cada espacio libre de la pila es un espacio nuevo para la introduccion de un nuevo empleado que consta de dos estructuras de datos.
El programa puede realizar las siguientes funciones:
 -Implementar una pila nueva con una base ya prederteminada.
 -Agregar a un empleado nuevo a la pila (Esta implementacion se realiza uno por uno).
 -Decir si la pila llego a tu limite o si se encuentra vacia.
 -Eliminar un empleado (Se elimina el ultimo trabajador agregado a la pila).
 -Mostrar la cantidad de empleados y sus datos que fueron agregados a la pila.
 -Limitar la cantidad de la pila si la persona lo desea.
